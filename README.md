# 📚 Wikipedia Assistant

This is a simple practice project where I built a small web app using Streamlit.
It lets you search any topic on Wikipedia, read a short summary, and download it as an audio file or PDF.

---

## 🚀 What it does

* Search any topic
* Shows a short summary
* Convert summary into:

  * 🎧 Audio (.mp3)
  * 📄 PDF (.pdf)
* Download the file easily

---

## 🛠️ Tech Used

* Python
* Streamlit
* wikipedia
* gTTS
* reportlab

---

## ▶️ How to run

```bash
python3 -m streamlit run app.py
```
